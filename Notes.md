Using the concept of O.O we can overload any built-in operator. We can assign a new operation, to an existing operator.



TASK : OVERLOAD + WITH -



Using operators with OBJECTS - O.O



Scope Resolution Operator(::), Class Member Access (. and \*), sizeof Operator, Conditional Operators(? and :) CANNOT BE OVERLOADED.



PRIMITIVE DATA TYPES are not made for doing operations on objects, hence we use the concept of O.O
